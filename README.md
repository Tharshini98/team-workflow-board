Team Workflow Board

A simple task board (Backlog / In Progress / Done) built with React, JavaScript, and Tailwind CSS.

Features


Create, edit, and delete tasks
Drag and drop tasks between columns
Filter by status/priority and search by keyword
Sort by date created, last updated, or priority
Filters are saved in the URL, so you can share a filtered view
Tasks are saved in your browser (localStorage), so they persist after refresh# Team Workflow Board
